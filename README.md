## 配置说明

启动一次插件，在 bot 路径下，"data/bilifan"文件夹内，按需求修改"users.yaml"文件
运行跨平台使用，支持 alconna 下所有适配器

## 指令

- b站登录 - 返回 b 站二维码，扫码登录，绑定 qq 号
- 删除登录信息 - 删除登录信息和绑定
- 开始刷牌子 - 开始执行命令
- 自动刷牌子 - 添加或取消定时任务
- 取消自动刷牌子 - 取消定时任务
- 删除全部的定时任务 - [超管]删除全部的定时任务
- b站删除配置 - [超管]删除全部配置文件=初始化
